설치하기

DoKev를 시작하기 위해 설치를 해봅시다!


다운로드

1. DoKev 레포지토리에 접속합니다.
2. 아래 이미지처럼 'Releases' 탭에서 가장 최신버전의 DoKev를 다운로드 합니다.

3. 자신의 운영체제에 맞는 DoKev를 다운로드 해주세요.
4. 압축을 해제합니다.


주의사항

- DoKev를 실행하기 위해서는 Python3가 설치되어 있어야합니다.
[ 이 곳에서 설치할 수 있습니다. ]

- 백신 소프트웨어가 DoKev의 파일 읽기/쓰기를 방해 할 수 있습니다. 이 점에 유의해주세요.

- DoKev에서 코드를 편리하게 작성하기 위해서는 Visual Studio Code가 필요합니다.
