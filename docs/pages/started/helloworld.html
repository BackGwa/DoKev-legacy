안녕, 세상아!

DoKev가 완벽하게, 작동하는지 "안녕, 세상아!"를 출력해봅시다!
HelloWorld.dkv 파일에는 기본적으로 아래와 같은 코드가 작성되어 있습니다.

- 안녕, 세상아! # 아래 코드는 "안녕, 세상아!"을 출력하는 코드입니다. ("안녕, 세상아!")라고 말해줘



Windows에서 실행하기

1. 명령 프롬프트에서 .\dokev.exe -c HelloWorld.dkv를 입력하여, 실행합니다.


macOS에서 실행하기

1. 터미널에서 ./dokev -c HelloWorld.dkv를 입력하여, 실행합니다.


Linux에서 실행하기

1. 터미널에서 ./dokev -c HelloWorld.dkv를 입력하여, 실행합니다.



실행 결과

성공적으로 빌드가 마무리되었다면, 아래와 같은 내용이 결과로 출력됩니다.

- 빌드 결과 안녕, 세상아!
