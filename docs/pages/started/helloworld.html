안녕, 세상아!

DoKev가 완벽하게, 작동하는지 "안녕, 세상아!"를 출력해봅시다!
source.dkv 파일에는 기본적으로 아래와 같은 코드가 작성되어 있습니다.

- 안녕, 세상아! # 아래 코드는 "안녕, 세상아!"을 출력하는 코드입니다. ("안녕, 세상아!")라고 말해줘



Windows에서 실행하기

1. Runner.exe를 실행합니다.


macOS에서 실행하기

1. DoKev 폴더의 위치에서 터미널을 엽니다.
2. 터미널에서 다음 명령을 실행합니다. chmod -x Runner
3. Runner를 실행합니다.


Linux에서 실행하기

1. Runner를 실행합니다.



실행 결과

성공적으로 빌드가 마무리되었다면, 아래와 같은 내용이 결과로 출력됩니다.

- 빌드 결과 안녕, 세상아!
