> [NOTE]

> 일부 문법과 호환되지 않습니다. C# 기반의 DoKev를 사용하고 싶다면, 해당 [문서](https://backgwa.notion.site/backgwa/DoKev-46bc63939be74aa28d3247ed2ec8d415)를 참고하세요!
